Registering for GradeScope

For new students:

  1. Open gradescope.com and sign up as a student.
  2. Find the button that says "Add Course", and then use the entry key {% if student.legit %} {{ semester.gradescope_key }}. {% else %} [redacted]. {% endif %}
  3. For school, choose Massachusetts Institute of Technology.
  4. Input your full name, with given name first and family name last.
  5. Choose any email address that you want. You'll get emails here notifying you when scores are available.
  6. For student ID, type {{ student.pk }}. You must type this exact number to receive spades for practice exams.

If you are a returning student, you should be able to just log in to your old account and then use the entry code above. (But you can create another new account if you want, too.)

Submitting to GradeScope

In brief:

  1. The time limit is 4.5 hours.
  2. Like a real olympiad, submit solutions on the answer sheets.
  3. Submit to GradeScope. Be careful to submit the right papers to the right buckets (you will only submit for three of five problems). Do not forget to assign pages to problems; grading is done on a per-problem basis so if you skip this important step then your submission will simply show as blank to every grader for every problem.
  4. Please submit at least one page for all three problems on your test.

Check the syllabus for full details.
